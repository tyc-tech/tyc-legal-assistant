# 知产资产清单

## 触发条件

科技型企业并购、知产质押融资、技术壁垒评估、知产资产证券化。

关键词：知产资产、专利、商标、软著、技术壁垒、IP

## 输入要求

- **searchKey** (必填): 企业名称 或 信用代码

## 执行流程

### Step 1: 知产存量
- `get_patent_info` — 专利
- `get_trademark_info` — 商标
- `get_software_copyright_info` — 软件著作权
- `get_copyright_work_info` — 作品著作权

### Step 2: 网络服务备案
- `get_internet_service_info` — ICP 备案 + 公众号

### Step 3: 创新力评估
- `get_ipr_score` — 创新力评分

### Step 4: 知产风险
- `get_ipr_pledge` — 知产出质（已质押 IP 警示）

### Step 5: 历史 IP（已过期/失效）
- `get_historical_trademark` — 历史商标
