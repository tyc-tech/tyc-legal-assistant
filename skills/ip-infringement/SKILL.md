# 知产侵权预警

## 触发条件

产品上市前、品牌推广前、合作签约前的知产侵权预防扫描。

关键词：知产侵权、商标近似、专利覆盖、品牌预警、IP clearance

## 输入要求

- **searchKey** (必填): 产品名称 / 品牌名 / 企业名称
- **applicant** (可选): 已持有专利/商标的申请人企业

## 执行流程

### Step 1: 商标近似检索
- `search_trademarks` (`searchKey`) — 同名/近似商标
- `get_trademark_detail` (`regNo`) — 关键近似商标详情

### Step 2: 专利覆盖检索
- `search_patents` (`searchKey`, `applicant`) — 关键技术专利

### Step 3: 自有 IP 状况（若提供 applicant）
- `get_trademark_info` — 申请人已持商标
- `get_patent_info` — 申请人已持专利

### Step 4: 知产质押风险
- `get_ipr_pledge` — 知产出质（识别已质押的核心 IP）

## 输出格式

```markdown
# 知产侵权预警 — {searchKey}

## 一、商标近似风险
| 近似商标 | 注册号 | 状态 | 持有人 | 国际分类 |
|---------|--------|------|--------|---------|

冲突评估:
- 完全相同商标: {n} 条
- 近似商标（1-3 字差异）: {n} 条
- 跨类别风险: ...

## 二、专利覆盖风险
| 关键专利 | 申请号 | 申请人 | 状态 |
|---------|--------|--------|------|

冲突评估:
- 直接覆盖: {n} 件
- 边缘覆盖: {n} 件

## 三、自有 IP 状况
（若提供 applicant）

## 四、规避建议
- 商标:
  - 完全更名 / 跨类申请 / 受让 / 共存协议
- 专利:
  - 设计绕开 / FTO 报告 / 专利许可

## 五、风险等级
- 商标侵权风险: 高 / 中 / 低
- 专利侵权风险: 高 / 中 / 低
- 综合建议: 推进 / 调整后推进 / 暂停
```

## 示例

输入: `searchKey = "新品牌名称"`, `applicant = "我方公司"`
