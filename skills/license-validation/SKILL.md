# 合规资质有效性验证

## 触发条件

合作方年度续约审查、采购供应商年度合规审查、资质类合作前置审核。

关键词：资质核验、合作续约、电信许可、行政许可、到期监控

## 输入要求

- **searchKey** (必填): 合作方企业名称 或 信用代码

## 执行流程

### Step 1: 主体核验
- `get_company_registration_info`

### Step 2: 资质证书全集
- `get_qualifications` — 资质证书（含等级 / 有效期 / 状态）
- `get_administrative_license` — 行政许可（决定文书号 / 有效期）
- `get_telecom_license` — 电信业务许可（业务类别 / 范围）

### Step 3: 历史失效（识别"曾经持有但已失效"）
- `get_historical_admin_license` — 历史行政许可（已过期）

### Step 4: 关联资产合法性
- `get_construction_qualifications` — 建筑资质（如适用）

## 输出格式

```markdown

## 一、当前有效资质
| 证书类型 | 证书编号 | 等级 | 发证日期 | 有效期至 | 距到期 |
|---------|---------|------|---------|---------|--------|

⚠️ **临期证书（< 90 天）**: {n} 张
🚨 **已过期证书**: {n} 张

## 二、行政许可
| 许可名称 | 决定文书号 | 许可机关 | 有效期至 |
|---------|----------|---------|---------|

## 三、电信业务许可
（业务类别 / 业务范围 / 发证日期 / 有效期）

## 四、建筑资质（如适用）
（资质类别 / 等级 / 项目能力）

## 五、历史失效资质（参考）
（曾持有但已过期的证书清单）

- 资质合规等级: A / B / C
- 是否可以续约: ✓ / ⚠️ 需补证 / ❌ 不建议
- 待补证清单: ...
- 监控重点: 临期证书 {n} 张
```
